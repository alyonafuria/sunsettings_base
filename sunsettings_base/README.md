# Sunsettings

A web application that combines sunset photography with AI-powered quality analysis and NFT minting capabilities. Users can analyze sunset quality at any location, upload sunset photos to IPFS, and mint them as NFTs on the Base blockchain.

**Live App**: [catch.sunsettings.app](https://catch.sunsettings.app)

## Features

- **Sunset Quality Analysis**: AI-powered scoring and descriptions for sunset conditions at any location
- **Interactive Map**: Visualize sunset photos on a global map with location pins
- **Photo Upload**: Store sunset photos on decentralized IPFS storage
- **NFT Minting**: Mint sunset photos as NFTs on the Base blockchain with gasless transactions
- **Smart Wallet Integration**: Seamless onboarding with Base Smart Wallet and Privy authentication
